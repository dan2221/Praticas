Descrição do curso

Este curso apresenta conceitos básicos de web e mostra as partes e funções das tags em um documento HTML. Ensina, também, formatar uma página, inserir imagens e muito mais! Tudo isso, com explicações simples e de fácil compreensão.

Estrutura do curso
  1. Módulo 1 - Introdução
  2. HTML e a web
  3. Criando um documento HTML
  4. Formatando um documento HTML
  5. Criando listas
  6. Inserindo imagens
  7. Criando links
  8. Combinando tags
  9. Usando imagens mapeadas

Não há pré-requisitos para a realização deste curso.
